What is this?

Testing is hard to learn; many students struggle in defining their own test cases and miss important cases.

This system provides users with a collection of scenarios that require writing certain test cases. They would then be challenged to find a test case that meets certain conditions. The conditions range from identifying an incorrect program apart from a fully functioning program, to reaching a particular place in the code.

How to play

Statement Mutation

developer cut and pastes a part of code of which the outcome may be removal of some lines

Value Mutation
